# OffAxis / Chat
Made with [Mern.io](http://mern.io/) (Mongo, ExpressJs, React, NodeJs), Redux, Socket.io and Bootstrap

## Functionnality
* Multilingual
* JWT Auth
* Users, Rooms, Messages management
* Room search and paging
* Live interractions :
    * Connected users
    * User is writting
    * User has new messages
